## 📖 Overview

**Python-NumericalComputingLab** is a collection of four Python laboratory programs that demonstrate fundamental concepts in numerical computing, mathematical analysis, data visualization, and financial modeling.

The repository provides hands-on examples using Python, NumPy, Matplotlib, and modular programming techniques.

---

## 🎯 Objectives

* Understand Python package creation and usage.
* Apply combinatorial mathematics concepts.
* Perform numerical computations using NumPy.
* Visualize mathematical and statistical data.
* Analyze datasets using descriptive statistics.
* Simulate financial market behavior using synthetic data.

## 🚀 Programs Included

### 1️⃣ Combinatorial Mathematics Package System

#### 📌 Description

Creates a reusable factorial package and utilizes it within another package to calculate:

* Factorial
* Combination (nCr)
* Permutation (nPr)

#### 🛠 Concepts Covered

* Python Packages
* Modular Programming
* Factorial Computation
* Combinations and Permutations

#### 📚 Libraries Used

* Python Standard Library

---

### 2️⃣ Mathematical Function Visualization

#### 📌 Description

Combines Python's built-in `math` module with NumPy to perform mathematical computations and visualize the results using Matplotlib.

#### 🛠 Operations Performed

* Exponential Function
* Logarithmic Function
* Trigonometric Function (Sine)
* Mathematical Plotting

#### 📚 Libraries Used

* math
* NumPy
* Matplotlib

---

### 3️⃣ Statistical Distribution Analysis

#### 📌 Description

Generates a dataset from a normal distribution and performs statistical analysis to study the characteristics of the data.

#### 🛠 Statistical Measures

* Mean
* Median
* Standard Deviation

#### 📊 Visualization

* Histogram of Normal Distribution

#### 📚 Libraries Used

* NumPy
* Matplotlib

---

### 4️⃣ Financial Market Simulation and Return Analysis

#### 📌 Description

Simulates stock market price movements using synthetic daily returns and evaluates financial performance metrics.

#### 🛠 Financial Metrics

* Daily Returns
* Stock Prices
* Average Return
* Volatility
* Cumulative Return

#### 📊 Visualizations

* Stock Price Movement
* Cumulative Return Trend

#### 📚 Libraries Used

* Matplotlib

## ⚙️ Installation

### Clone the Repository

```bash
git clone https://github.com/<your-username>/Python-NumericalComputingLab.git
```

### Navigate to the Project Directory

```bash
cd Python-NumericalComputingLab
```

### Install Required Libraries

```bash
pip install -r requirements.txt
```
